A proposta desse projeto é criar uma página web estilizada e que exibe listas. 

Para isso utilizei as linguagens HTML e CSS. 

Há um arquivo HTML que conterá a estrutura da página da imobiliária e um arquivo CSS que conterá os estilos para mudar a aparência da página.

Ao criar esse projeto, coloquei em prática os seguintes conceitos:

- Tags básicas do HTML
- Meta tags HTML
- Listas no HTML
- Seletores CSS
